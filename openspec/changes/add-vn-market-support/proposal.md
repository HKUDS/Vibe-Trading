# Proposal: Add Vietnam Stock Market Support

> Generated by: `cm-brainstorm-idea` → `cm-planning`
> Date: 2026-05-10
> Status: DRAFT — pending owner approval
> Initiative: `add-vn-market-support`

## Why

Vibe-Trading hiện cover 6 thị trường (A-shares, HK, US, Crypto, China Futures, Global Futures, Forex) nhưng **bỏ trống hoàn toàn HOSE/HNX/UPCoM**. Trong khi đó:

- TTCK Việt Nam đang trên đường **nâng hạng FTSE EM 2025-2026** và **KRX go-live 2025**
- ~2 triệu nhà đầu tư active, 50k+ tài khoản mở mới/tháng (số liệu VSDC 2025)
- Không tool AI-driven nào trên thị trường hiểu đặc thù VN: T+2, biên độ ±7/10/15%, lô 100, thuế 0.1% bán, báo cáo VAS, VN30F, room ngoại, GDKHQ
- A-share engine **không thể tái dùng** vì T+1≠T+2, ticker format khác (`VNM` vs `000001.SZ`), biên độ khác, broker schema khác

## What Changes

Thêm Vietnam như một **first-class market** ngang hàng A-shares, theo nguyên tắc kiến trúc đã có (provider-agnostic, hot-load skills, composite engine).

### High-level deltas

| Domain | Delta |
|--------|-------|
| `data-sources` | + `VNStockProvider` (vnstock + TCBS + SSI + DNSE fallback chain), + `VNFundamentalProvider` (VAS PIT) |
| `backtest` | + `VNEquityEngine` (T+2, biên độ, lô 100, thuế+phí), + `VNFuturesEngine` (VN30F1M/F2M/F1Q/F2Q) |
| `skills` | +8 skills VN, mở rộng category `Asset Class` và `Risk Analysis` |
| `swarm` | +3 presets: `vn_investment_committee`, `vn_derivatives_desk`, `vn_value_screener` |
| `trade-journal` | + readers cho SSI, HSC, VNDirect, TCBS, DNSE |
| `strategy-export` | + Amibroker AFL exporter (giữ Pine v6) |
| (cross-cutting) | i18n locale `vi-VN` cho report, UBCK compliance disclaimer |

### In-scope
- Cổ phiếu HOSE/HNX/UPCoM (T+2, biên độ động, lô 100)
- Phái sinh VN30F (4 hợp đồng), ký quỹ + daily mark-to-market
- 5 broker journal formats phổ biến nhất
- Báo cáo backtest tiếng Việt + thuế/phí broker VN
- Amibroker AFL export (định dạng phổ biến nhất ở VN retail)

### Out-of-scope (future)
- ETF VN chuyên sâu (sẽ tận dụng `etf-analysis` skill chung)
- Trái phiếu doanh nghiệp / chứng quyền có đảm bảo (CW)
- Live order routing tới broker VN (vẫn là non-goal toàn dự án)
- Tích hợp Zalo OA / VNPay (thuộc về phạm vi business, không phải core)

## Approach

Reuse 3 trục kiến trúc sẵn có:

1. **DataProvider contract** → triển khai 2 provider mới, đăng ký vào fallback chain trong `data-routing` skill
2. **Engine interface** (`load_data`, `run`, `metrics`, `report`) → 2 engine mới, plug vào `CompositeEngine` cho danh mục đa thị trường (VN + Crypto, VN + US…)
3. **Skill registry hot-load** → +8 skills không cần đụng harness

i18n đi qua hệ thống report template hiện tại — thêm locale `vi-VN`, không thay logic.

## Risks & Mitigations

| Risk | Mitigation |
|------|------------|
| `vnstock` breaking changes (lib bên thứ 3 đang phát triển) | Pin version trong `pyproject.toml`, regression test theo Bundled-Skill Coverage |
| TCBS/SSI/DNSE đổi API public | Provider chain auto-fallback, mỗi provider có `is_available` health check |
| Sai sót trong logic biên độ/T+2 → backtest sai → user mất tiền thật | Unit test ma trận biên độ + T+2 settlement, regression theo từng broker, disclaimer UBCK rõ ràng |
| Báo cáo VAS khác IFRS, công thức factor có thể lệch | `vn-financial-statements-vas` skill chuyển đổi rõ ràng, document mapping VAS↔IFRS |
| Maintain thêm 1 market line tăng cost | Tận dụng tối đa code chung, chỉ override cái thực sự khác biệt |

## Roadmap (3 sprints, ~7-10 tuần)

- **Sprint 1 (2 tuần) — Core Equity:** `VNStockProvider` + `VNEquityEngine` + 3 skills cốt lõi (`vn-data-routing`, `vn-foreign-room`, `vn-financial-statements-vas`). Ship được backtest cổ phiếu VN end-to-end.
- **Sprint 2 (3 tuần) — Derivatives + Journal + Swarm:** `VNFuturesEngine` VN30F, 5 journal readers, 3 swarm presets, i18n `vi-VN`.
- **Sprint 3 (2-3 tuần) — Export + Advanced + Compliance:** Amibroker AFL, 5 skills nâng cao (`vn-ex-rights-calendar`, `vn-margin-list`, `vn-vn30-arbitrage`, `vn-sector-rotation-vn`, `vn-pre-warning-stocks`), UBCK disclaimer + audit pass.

## Success Criteria

1. User chạy `vibe-trading run -p "Backtest 20/50 MA crossover trên VNM, 1 năm"` → engine VN selected, vnstock fetch data, report tiếng Việt với thuế 0.1% và phí broker
2. `vibe-trading --swarm-run vn_investment_committee '{"topic":"FPT Q2"}'` → swarm chạy với báo cáo VAS PIT
3. User import journal SSI CSV → trade-journal analyzer tạo trading profile + 4 bias diagnostics
4. `/pine <run_id>` → emit thêm `strategy.afl` cùng với `.pine`, `.tdx`, `.mq5`
5. CI có ≥1 regression test cho mỗi skill VN mới và mỗi broker journal format
6. Zero-config free tier: user không cần API key vẫn backtest được VN equity (vnstock + TCBS public)

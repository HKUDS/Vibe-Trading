# Domain Glossary

> Bilingual domain terminology. Code APIs use English only. User-facing copy is bilingual.

| Chinese | English | Definition | Code Reference |
|---------|---------|------------|----------------|
| 策略 | Strategy | A complete trading logic with entry/exit/position rules, compiled from a template | `Strategy` entity |
| 策略模板 | Strategy Template | Pre-built strategy archetype with configurable parameters. Users fill params, not write code | `StrategyTemplate` |
| 回测 | Backtest | Historical simulation of a strategy's performance on past market data | `BacktestResult` |
| 跟单 | Copy Trade | Automatically mirroring another strategy's trading signals proportionally | `CopyTrade` entity |
| 信号 | Signal | A buy/sell/hold instruction generated by a strategy at a specific time | `Signal` value object |
| 信号源 | Signal Source | The creator user whose published strategy generates signals | `Strategy.creator_id` |
| 跟随者 | Follower | User who subscribes to and copies a published strategy | `CopyTrade.follower_id` |
| 因子 | Factor / Alpha | A quantitative indicator used in strategy construction (e.g., RSI, MACD) | `Factor` value object |
| 夏普比率 | Sharpe Ratio | Risk-adjusted return: (return - risk_free) / volatility | `BacktestResult.sharpe_ratio` |
| 最大回撤 | Max Drawdown | Largest peak-to-trough decline in portfolio value | `BacktestResult.max_drawdown` |
| 累计收益 | Cumulative Return | Total percentage return over the backtest period | `BacktestResult.cumulative_return` |
| 胜率 | Win Rate | Percentage of trades that were profitable | `BacktestResult.win_rate` |
| 仓位 | Position Size | Percentage of capital allocated to a single trade | `StrategyParams.position_size` |
| 止损 | Stop Loss | Automatic exit when loss exceeds a threshold | `StrategyParams.stop_loss` |
| 止盈 | Take Profit | Automatic exit when profit reaches a target | `StrategyParams.take_profit` |
| 均线交叉 | Moving Average Cross | Template: enter when short MA crosses above long MA | `MovingAverageCrossTemplate` |
| 网格交易 | Grid Trading | Template: place orders at fixed price intervals in a range | `GridTradingTemplate` |
| RSI 超买超卖 | RSI Mean Reversion | Template: buy when RSI < 30, sell when RSI > 70 | `RSIMeanReversionTemplate` |
| 突破追涨 | Breakout Momentum | Template: enter on price breakout above resistance | `BreakoutMomentumTemplate` |

## Naming Conventions

- **Entities** (have identity): `Strategy`, `CopyTrade`, `User` — nouns, singular
- **Value Objects** (no identity): `Money`, `Percentage`, `Signal`, `TimeRange` — nouns, immutable
- **Repositories**: `StrategyRepository`, `CopyTradeRepository` — `[Entity]Repository`
- **Use Cases**: `CreateStrategy`, `FollowStrategy`, `RunBacktest` — verb phrases, imperative
- **DTOs**: `CreateStrategyRequest`, `StrategyResponse`, `BacktestResultDTO` — suffix `Request`/`Response`/`DTO`
- **Events**: `StrategyPublished`, `SignalGenerated` — past tense verb phrase
